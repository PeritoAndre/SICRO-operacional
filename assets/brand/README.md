# Brand assets

Place the institutional logos used by the duty report here.

Preferred file names:

- `governo_amapa.png`
- `policia_cientifica.png`

The app also accepts `.jpg` and `.jpeg` with the same base names, but
transparent PNG is preferred.

Recommended source files:

- Governo do Amapa: transparent PNG, about 900x360 px.
- Policia Cientifica: transparent PNG, square, about 600x600 px.

The PDF service loads these files when available and falls back to text boxes
while the images are absent.
